# Random Quote Generator

A simple web application that generates random quotes using an API. The app displays a random quote along with the authorâ€™s name, and users can fetch a new quote by clicking a button. The design is clean and user-friendly.


## Features

- Fetches random quotes from the [Quotable API](https://api.quotable.io/quotes/random).
- Displays quotes with author names.
- "Loading" animation while fetching a new quote.
- Error handling for API requests.
